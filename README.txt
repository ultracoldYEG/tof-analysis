For this package to run correctly 
1. Install the Anaconda distribution on Windows for Python 2.7
2. Download and install Microsoft Visual C++ Compiler for Python 2.7: https://www.microsoft.com/en-ca/download/details.aspx?id=44266
3. After creating a runTOF.bat file, change the path to the "activate.bat" file in the local /Anaconda2/Scripts directory
4. Change the save file paths in tof-analysis.py
